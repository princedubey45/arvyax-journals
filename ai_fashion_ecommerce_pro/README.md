
AI FASHION ECOMMERCE PROJECT

This is a professional full‑stack starter for building
a Myntra‑like AI powered ecommerce platform.

Main Features
- Authentication system
- Product catalog
- Cart
- Checkout
- Reviews
- AI recommendation engine
- AI stylist chatbot
- Image search
- Virtual try on

Run Project

Backend
cd backend
npm install
npm run dev

Frontend
cd frontend
npm install
npm run dev

Frontend runs on http://localhost:3000
Backend runs on http://localhost:5000
